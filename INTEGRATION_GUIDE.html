Cookie Banner & Legal Footer

Nordic Anchor — Integration Guide · Apply to all 5 existing pages

Overview

This guide adds a GDPR-compliant cookie consent banner (slides up from the bottom on first visit) and adds Privacy Policy / Terms / Cookie Settings links to every page footer. The two new pages — /privacy and /terms — are already built and provided as separate files ready to drop into your repo.

Find the opening <body> tag. Paste this block immediately after it, before the nav-overlay div.

HTML to paste

4 Add JavaScript — just before </body>

Paste this script block at the bottom of each page just before </body>. If there is already a <script> block there, paste just the functions inside it rather than adding a second script tag.

JavaScript to paste
<script>
  function cookieAccept(){
    localStorage.setItem('na_cookie_consent','accepted');
    hideBanner();
  }
  function cookieDecline(){
    localStorage.setItem('na_cookie_consent','declined');
    hideBanner();
  }
  function hideBanner(){
    document.getElementById('cookieBanner').classList.remove('visible');
  }
  function resetCookieConsent(){
    localStorage.removeItem('na_cookie_consent');
    document.getElementById('cookieBanner').classList.add('visible');
  }
  (function(){
    if(!localStorage.getItem('na_cookie_consent')){
      setTimeout(()=>document.getElementById('cookieBanner').classList.add('visible'),800);
    }
  })();
</script>

5 vercel.json — no changes needed ✓

Because cleanUrls: true is already set in your vercel.json, placing privacy.html and terms.html in the repo root is all that's needed. They will automatically serve at /privacy and /terms. No config changes required.
